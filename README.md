# Leviathan website

Marketing site for [Leviathan](https://x.com/leviathanfront): trustless training for the people's model.

## Stack

Vite, React 19, Tailwind v4, Motion.

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
